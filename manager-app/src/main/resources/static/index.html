Добро пожаловать в StockMaster

StockMaster — веб-приложение для управления ассортиментом товаров.

Позволяет менеджеру:

  • просматривать товары
  • добавлять новые позиции
  • редактировать информацию
  • удалять устаревшие товары

Всё это — через простой и удобный веб-интерфейс.
